namespace
*********

By using namespace we can do cluster (dividing)partitions and you can run containers in isolated environments


$ kubectl get namespace---> used to check namespaces 

$ kubectl get namespace | wc -l---> used to check namespaces with line count 

$ kubectl get pods -n(namespace) research --> used to find number of pods
